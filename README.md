Please make sure you have a suitable C compiler (like GCC) that supports C99 or newer in order to run this program.

To run this program, follow the steps given below (assuming you are on Windows and using gcc).
1. Download the movieTicketManagementSystem.c file and place it in any folder you want (a dedicated folder is recommended).
2. Open the folder with the file and type cmd in the address bar. This will open a command prompt in that directory.
3. Type "gcc movieTicketManagementSystem.c -o movieTicketManagementSystem.exe" in the console. This will compile your program and create an executable file.
4. Now, type ".\movieTicketManagementSystem.exe", and voila! You are good to go.
